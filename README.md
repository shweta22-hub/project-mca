# 💰 AI-Based Cloud Cost Anomaly Detection System

## 📌 Overview
This project uses Machine Learning to detect unusual spikes in cloud costs and generate alerts.  
It helps in identifying unexpected increases in usage and optimizing cloud expenses.

---

## 🎯 Objective
- Monitor cloud cost data
- Detect anomalies using AI
- Provide alerts for unusual spending patterns

---

## 🚀 Features
- 📊 Cost visualization dashboard
- 🤖 Anomaly detection using Isolation Forest
- 🚨 Highlight unusual cost spikes
- 📁 Simple and clean dataset handling

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

---

## 📊 How It Works
1. Load cloud cost dataset
2. Preprocess data
3. Train anomaly detection model
4. Identify unusual patterns
5. Display results on dashboard
